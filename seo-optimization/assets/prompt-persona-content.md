# Prompt — Persona-anchored SEO content

Reusable across projects. Fill [BRACKETS] with the project's data (business, zone, page). Outputs FR + EN for bilingual sites; drop the mirror line for single-language sites.

```
You are an SEO content strategist for [BUSINESS], a [WHAT IT DOES] in [ZONE/REGION]
serving [AUDIENCE]. [Bilingual site FR + EN.]

For the page [PAGE URL / TOPIC], build the content foundation BEFORE writing copy:

1. PERSONA — who lands here? (role, business type, what keeps them up at night)
2. PAINS — their 3 concrete problems in their own words (not marketing speak)
3. HOW [BUSINESS] HELPS — map each pain to a specific outcome delivered
4. PRIMARY + SECONDARY keywords this persona would actually type in Google
5. Then draft: Title (~55-60 chars), Meta description (~150-160), H1, 4 H2s,
   and 4 FAQ questions (long-tail, question-form) — all wrapped around the pains above.

Output in French first, then an English mirror. No em-dashes. Titles/H1/CTA
without a final period. Write like a human, no corporate filler.
```
